Ajax AutoSearch is a free WordPress Search Plugin that comes with a handful of essential customization options to enhance the search engine feature.

== Description ==
Ajax AutoSearch is a free WordPress Search Plugin that enhances the Search feature on your site. The plugin provides a handful of crucial customization options with which you have full control over the search results. Ajax AutoSearch includes features like Different Search Filters, Display Featured Image, Post Limit, and more. To make things easier for your visitors, you can display the search filter options like post, page, categories, and tags. Your visitors can choose the search content type they are searching for and get the adequate results in just a few seconds with the search filters option available. You can also choose to display the featured images of the search results. Ajax AutoSearch allows you to display up to 20 search results on a single page. The plugin is extremely lightweight and comes with only the essential configuration options. So, improve the search results for your visitors with Ajax AutoSearch. A quick solution for better user experience should always be a priority for all webmasters.

== Installation ==
The easy way (via Dashboard) :

* Go to Plugins > Add New
* Type in the **Ajax AutoSearch** in Search Plugins box
* Click Install Now to install the plugin
* After Installation click activate to start using the **Ajax AutoSearch**
* Go to **Ajax AutoSearch** from Dashboard menu

Not so easy way (via FTP) :

* Download the **Ajax AutoSearch**
* Unarchive **Ajax AutoSearch** plugin
* Copy folder with ajax-autosearch.zip
* Open the ftp \wp-content\plugins\
* Paste the plug-ins folder in the folder
* Go to admin panel => open item "Plugins" => activate **Ajax AutoSearch**
* Go to **Ajax AutoSearch** from Dashboard menu
